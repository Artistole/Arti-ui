Avatar with text

Text is used when an image is not available. This is usually done by taking the first letter of first name and last name of the user

Avatar with text component is available in 3 different sizes.

Include class avatar and according to the required size add class avatar-lg or avatar-md or avatar-sm.

Now use the class avatar-txt to apply the styling for the text

(e.g. class="avatar avatar-lg avatar-txt")
