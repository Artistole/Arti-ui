Badge
A badge is a counter shown on icons like cart, notification and avatars. It is also used to show online and offline status of a person.
Badge on avatars
Badge on avatar can be used to show the current status of a user. It can be used to show whether the user is online or offline or busy or away.
code snippet:
You can use the previous avatar classes for avatar and then use class badge and badge-online or badge-offline or badge-away or badge-busy as per the requirement.
Badge on icons
Number Badge is used to show the no. of items present in the parent component.
