To use these alerts just use class name alert and add id according to alert type: scl-alert-primary or scl-alert-success or scl-alert-warning or scl-alert-error.
You can also add icons as per your need using id according to icon type: scl-icon-primary or scl-icon-success or scl-icon-warning or scl-icon-error.
